# Overview
DIFMS is a web-based platform designed to help small and medium-sized businesses digitise their invoicing and basic financial management processes. The system provides a simple and accessible way to create invoices, monitor payment statuses, manage customer records, and view basic financial insights.
The platform focuses on simplicity, fast adoption, and practical use in everyday business operations, especially for users moving from manual or spreadsheet-based systems.

# Core Features
Customer management (add, edit, track clients)
Invoice creation with itemised entries
Payment status tracking (paid, unpaid, overdue)
PDF invoice generation and export
Basic dashboard for revenue and invoice insights
